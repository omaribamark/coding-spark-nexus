🛡️ HAKIKISHA Backend API

✅ Backend Server Running

This is a Node.js/Express backend API for the HAKIKISHA mobile application.

⚠️ Important Note

This backend must be deployed to AWS to work with your mobile app. It cannot run in Lovable's web preview.

🚀 Deployment

Follow the instructions in BACKEND_DEPLOYMENT.md to deploy to AWS.

📚 API Documentation

Base URL: https://your-aws-domain.com/api/v1

✨ Key Features Implemented:

  • ✅ Auto-created admin with 2FA authentication
  • ✅ Points system with daily reset (skip a day = reset to 0)
  • ✅ JWT Authentication with refresh tokens
  • ✅ User, Fact-Checker, and Admin roles
  • ✅ Claims submission and verification
  • ✅ AI-assisted fact-checking
  • ✅ Blog management
  • ✅ Leaderboard and streak tracking
  • ✅ CORS configured for mobile apps

📱 Mobile App Integration

Update your mobile app's API base URL to point to your deployed AWS backend:

API_BASE_URL = "https://your-aws-domain.com/api/v1"

🎯 Points System

  • Daily Login: 5 points
  • Submit Claim: 10 points
  • First Claim: 50 points
  • Claim Verified: 20 points
  • ⚠️ Skip a day? Points reset to ZERO!
  • 🔥 Streak bonuses at 3, 7, and 30 days
